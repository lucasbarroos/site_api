# Vale Lounge Api
Website Api with Nodejs and Mongodb Atlas

# Useful Commands
To run the development server:
```
yarn
yarn dev
```
To make the production server with [PM2](https://pm2.keymetrics.io/):
```
yarn 
yarn build
yarn production
```
To generate a new documentation using [ApiDoc](https://apidocjs.com/):
```
yarn
yarn doc
```

# Technologies
```
Nodejs
Mongodb
Mongodb Atlas

ApiDoc
```
